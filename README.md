# 农夫乐事扩展书汉化  
如果你是来下载mod的，请直接跳到release。  
如果你想要更高版本，请看[fd-cookbook-reforged](https://github.com/Hotakus/fd-cookbook-reforged)。  

## 文件说明  
此项目仅存档汉化文件和数据信息，汉化是通过直接往jar包中放文件完成的，没有编译过程。  
- `food_data`: 此目录下存放农夫乐事在1.16.5和1.18.2的食物数据表，同时提供了一个Python脚本用于自动查询。
- `replace_file`: 此目录下存放替换的汉化文件。
- `scratches`: 这个汉化是一位群友的主意，此目录下存放她的初稿。
